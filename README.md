# 📊 Content Production Hub

কন্টেন্ট টিমের জন্য একটি ডায়নামিক ওয়েবসাইট — অ্যাডমিন প্যানেল (তুমি নিয়ন্ত্রণ করবে) + কর্মীদের ডেটা-এন্ট্রি ইন্টারফেস + গ্যাপ-অ্যানালাইসিস ড্যাশবোর্ড।

---

## ✨ কী কী আছে

- **ড্যাশবোর্ড** — এক নজরে: মোট কন্টেন্ট, পাবলিশড, বাকি, শেষ ৭ দিনের পোস্ট। হেড ও টাইপ অনুযায়ী **গ্যাপ** রঙে দেখায় (✅ ঠিক আছে / ⚠️ গ্যাপ / 🟠 পোস্ট বাকি / 🔴 নেই)।
- **কন্টেন্ট এন্ট্রি** — কর্মীরা প্রতিদিন কন্টেন্ট এন্ট্রি দেয়; এক্সিকিউটিভরা পোস্ট তারিখ, প্ল্যাটফর্ম ও লিংক যোগ করে।
- **অ্যাডমিন প্যানেল (শুধু তুমি)**
  - 👥 **ইউজার ও অ্যাক্সেস** — কর্মী যোগ করা, প্রতিজনকে আলাদা অনুমতি দেওয়া (কন্টেন্ট তৈরি / পোস্ট / অ্যাডমিন), একাউন্ট চালু-বন্ধ।
  - 🗂️ **কাস্টমাইজেশন** — কন্টেন্ট হেড, টাইপ ও প্ল্যাটফর্ম যোগ/মুছে ফেলা।
  - ⚙️ **সেটিংস** — সাইটের নাম, গ্যাপ অ্যালার্টের দিন সংখ্যা।

---

## ▶️ চালানোর নিয়ম (লোকাল কম্পিউটারে)

১. কম্পিউটারে **Node.js** ইনস্টল থাকতে হবে (https://nodejs.org — LTS ভার্সন)।
২. এই ফোল্ডারটি খুলে টার্মিনাল/কমান্ড প্রম্পটে গিয়ে লিখুন:

```bash
node server.js
```

> প্রথমবার কাজ না করলে আগে একবার: `npm install` লিখুন, তারপর `node server.js`।

৩. ব্রাউজারে যান: **http://localhost:3000**

### প্রথম লগইন
- ইমেইল: **admin@example.com**
- পাসওয়ার্ড: **admin123**

> ⚠️ ঢুকেই **অ্যাডমিন → ইউজার ও অ্যাক্সেস**-এ গিয়ে নিজেকে এডিট করে পাসওয়ার্ড বদলে নিন।

নমুনা কর্মী একাউন্ট: `rabbi@example.com / 1234` (শুধু তৈরি), `tanvir@example.com / 1234` (শুধু পোস্ট)।

## 📧 ইমেইল দিয়ে অ্যাক্সেস

- প্রতিটি কর্মীর একটা **ইমেইল** থাকে — সেটাই তাদের লগইন আইডি।
- অ্যাডমিন **ইউজার ও অ্যাক্সেস**-এ নতুন কর্মী যোগ করার সময় নাম, ইমেইল, পাসওয়ার্ড ও অনুমতি দেয়।
- চাইলে **"লগইন তথ্য এই ইমেইলে পাঠাও"** টিক দিলে কর্মীর ইমেইলে লগইন তথ্য চলে যাবে।

**ইমেইল পাঠাতে একবার সেটআপ:** অ্যাডমিন → ⚙️ সেটিংস → *ইমেইল (SMTP)*-এ একটা ইমেইল অ্যাকাউন্টের তথ্য দিন।
- Gmail হলে: হোস্ট `smtp.gmail.com`, পোর্ট `587`, ইউজার আপনার Gmail, পাসওয়ার্ডে **App Password** (Google Account → Security → App passwords থেকে তৈরি — সাধারণ পাসওয়ার্ড নয়)।
- সাইটের লিংক (URL) দিলে ইনভাইট ইমেইলে সেই লিংকও যাবে।

> SMTP সেট না থাকলেও সমস্যা নেই — অ্যাকাউন্ট তৈরি হবে, শুধু আপনি নিজে কর্মীকে ইমেইল/পাসওয়ার্ড জানিয়ে দেবেন।

## 🌐 ইন্টারনেটে লাইভ করতে (যাতে কর্মীরা যেকোনো জায়গা থেকে ঢুকতে পারে)

যেকোনো Node.js হোস্টিং-এ আপলোড করলেই চলবে। সহজ ও ফ্রি অপশন:

- **Render.com** বা **Railway.app** — নতুন Web Service খুলে এই ফোল্ডার আপলোড করুন।
  - Build Command: `npm install`
  - Start Command: `node server.js`
  - হোস্ট নিজে থেকে একটা পোর্ট দেয় — কোড সেটি স্বয়ংক্রিয়ভাবে ব্যবহার করে (`PORT`)।

একজন ডেভেলপারকে দিলে ৫–১০ মিনিটেই লাইভ করে দিতে পারবে।

## 💾 ডেটা কোথায় থাকে

সব তথ্য `data/store.json` ফাইলে জমা হয়। **এই ফাইলটি ব্যাকআপ রাখলেই পুরো ডেটার ব্যাকআপ হয়ে যায়।** হোস্টিং বদলালে শুধু এই ফাইল কপি করলেই ডেটা সাথে যাবে।
